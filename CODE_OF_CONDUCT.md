# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, attack problems rather than people,
and keep criticism specific and technical. Receipts about code are welcome;
receipts about humans are not.

Unacceptable behavior can be reported to the maintainer via GitHub
(@huyn7539) or through a private security advisory on this repository if
discretion is needed. Reports are handled confidentially.

Enforcement follows the Contributor Covenant's enforcement guidelines:
correction → warning → temporary ban → permanent ban, proportionate to the
violation.
